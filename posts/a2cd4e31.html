win10常用快捷键

发表于2022-03-02|更新于2022-06-09|教程
|字数总计:893|阅读时长:3分钟|阅读量:

Windows徽标键

快捷键 功能描述
win + D 显示桌面
win + E 打开文件资源管理器
win + I 打开设置
win + L 锁屏
win + M 最小化所有窗口
win + R 打开运行
win + S 打开搜素
win + T 任务栏切换应用预览
win + V 打开剪切板
Ctrl + W 关闭活动窗口
win + X 打开快捷菜单
win + 逗号 临时快速查看桌面
win + Pause 打开关于电脑
win + 数字 打开任务栏位于该数字所表示位置的应用
win + Tab 打开任务视图
win + ↑ 最大化窗口
win + ↓ 最小化窗口
win + ← 最大化窗口到屏幕左侧
win + → 最大化窗口到屏幕右侧
win + Home 除当前活动窗口，其他窗口最小化
win + 空格 切换不同输入法
win + + 使用放大镜
win + Alt + D 显示日历
win + Shift + ↑ 窗口向顶部和底部拉伸
win + Ctrl + D 添加虚拟桌面
win + Shift + → 切换至右侧虚拟桌面
win + Shift + ← 切换至左侧虚拟桌面
win + Shift + F4 关闭当前虚拟桌面

Ctrl键

快捷键 功能描述
Ctrl + A 选择文档或窗口中的所有项目
Ctrl + C 复制选定文本
Ctrl + D 删除选定项至到回收站
Ctrl + E 选择页面中的搜索框
Ctrl + F 在页面上查找
Ctrl + H 打开历史记录
Ctrl + N 打开新窗口
Ctrl + O 打开”打开文件”
Ctrl + P 打印当前页面
Ctrl + S 保存当前操作的文件
Ctrl + T 新建标签
Ctrl + V 粘贴选定文本
Ctrl + W 关闭活动窗口
Ctrl + X 剪切选定文本
Ctrl + Y 恢复撤销操作
Ctrl + Z 撤消操作
Ctrl + 鼠标滚轮 更改桌面图标大小
Ctrl + Home 移至编辑文本开头
Ctrl + End 移至编辑文本结尾
Ctrl + Tab 移动选项卡
Ctrl + 数字 移动到第 n 个选项卡
Ctrl + Shift + Esc 打开任务管理器
Ctrl + Shift + N 创建一个新文件夹
Ctrl + Shift + T 恢复最近关闭的选项卡
Ctrl + Alt + Delete 打开关闭程序对话框

文件资源管理器

快捷键 功能描述
Alt + → 后退
Alt + ← 前进
Alt + D 选择地址栏
Alt + Enter 打开选定项的属性
Ctrl + N 打开新窗口
Ctrl + W 关闭活动窗口
Ctrl + 鼠标滚轮 更改文件和文件夹图标的大小及外观
Backspace 查看上一级文件夹
Home 活动窗口定位至第一个文件
End 活动窗口定位至最后一个文件
F11 窗口最大化
Page Down 向下滚动
Page Up 向上滚动

浏览器

快捷键 功能描述
Ctrl + B 打开书签菜单
Ctrl + D 添加当前网页至收藏夹
Ctrl + J 打开下载历史记录
Ctrl + R 刷新当前标签
Ctrl + U 查看源代码
Ctrl + 鼠标滚轮 更改浏览器页面视图大小
Ctrl + + 放大 (25%)
Ctrl + - 缩小 (25%)
Ctrl + Shift + P 打开隐私窗口
Ctrl + Shift + Delete 打开清除浏览数据窗口
Shift + Space 向上滚动
Space 向下滚动
F5 刷新
F12 打开开发人员工具

文本编辑

快捷键 功能描述
Home 光标移动到当前行的行首
End 光标移动到当前行的行尾
Ctrl + 1 设置单倍行距
Ctrl + 2 设置双倍行距
Ctrl + B 所选文本设置为粗体
Ctrl + I 所选文本设置为斜体
Ctrl + U 所选文本添加下划线
Ctrl + H 所选文本替换
Ctrl + p 打印文档
Ctrl + = 所选文本添加下标
Ctrl + Shift + = 所选文本添加上标
Ctrl + Shift + > 增加字体大小
Ctrl + Shift + < 减小字体大小
F12 文本存为新文件

其他

快捷键 功能描述
Delete 删除选定项至到回收站
Shift + Delete 直接删除选定项
Print Screen 复制窗口截图到剪贴板
